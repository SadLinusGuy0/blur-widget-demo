# Changelog

All notable changes to Blur Widget Demo are documented here.

## [1.1.0] - 2026-06-30

### Added

- Added Samsung lock-screen widget support in 1x1 and 2x1 sizes.
- Added short randomized lock-screen messages such as `Glass`, `Calm`, and `Breathe`.
- Added Samsung monotone provider metadata for lock/AOD and cover surfaces.
- Added a ServiceBox RemoteViews receiver for Samsung SystemUI lock-screen requests.
- Added GitHub Wiki setup guides for One UI blur integration and lock-screen widgets.

### Changed

- Bumped the Android app version to `1.1.0`.

## [1.0.0] - 2026-06-29

### Added

- Initial release of the One UI Home blur widget demo.
- Added a resizable home-screen widget with adjustable tint and opacity.
- Added static widget picker previews and Samsung One UI widget metadata.
- Added signed release APK publishing through GitHub Actions.
